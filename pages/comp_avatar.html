Avatar

Avatar is a picture or component used to identify a user.
Use classes in format "avatar-size" to get avatar of required size. For example - "avatar-sm" for small size avatar.

<img src="https://picsum.photos/72" alt="avatar extra large" class="avatar-xl">

Rounded Avatar

Use classes in format "avatar-round-size" to get avatar of required size with rounded borders. For example - "avatar-round-sm" for small size avatar.

Avatar with status icon

Avatar icon with different background can be achieved by using following classes:

  • avatar-icon-primary
  • avatar-icon-secondary
  • avatar-icon-success
  • avatar-icon-caution
  • avatar-icon-error
